# password-strength-analyzer
A clean, security-focused desktop application that evaluates password strength using modern best-practice criteria.

Features:
- Real-time password analysis
- GUI-based interface (Tkinter)
- Clear strength scoring and feedback
- No password storage or logging

Strength Criteria:
- Length (12+ characters recommended)
- Uppercase and lowercase letters
- Numbers
- Special characters
- Detection of weak patterns

Use Case:
- IT Support
- Cybersecurity students
- Password hygiene training
- Desktop security utilities

How to Run:
```bash
python password_checker_gui.py
